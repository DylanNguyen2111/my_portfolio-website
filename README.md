## How to Run Locally

Clone the repository:

```bash
git clone https://github.com/DylanNguyen2111/portfolio-website.git
```

Open the folder:

```bash
cd portfolio-website
```

Open `index.html` directly in your browser, or use the **Live Server** extension in VS Code.

## Deployment

This site can be deployed using GitHub Pages:

1. Go to the repository settings.
2. Open **Pages**.
3. Select the `main` branch.
4. Save the settings.
5. GitHub will generate a live website link.
